# My Git Project
This repository contains basic Git version control practice.

## 📌 Purpose
- Practice Git commands
- Understand version control workflow
- Learn branching and merging

## 🛠️ Tools Used
- Git
- GitHub
- Git Bash

## 🚀 Basic Git Workflow
```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin <repository-url>
git push -u origin main
```

## 📁 Files
- `sampleetext.txt` — used to demonstrate all Git operations

## 🌿 Branches
- `main` — final merged version
- `secondary` — added a new line
- `merge-conflict` — used to resolve a merge conflict
